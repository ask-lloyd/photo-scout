08

UX Guide

Design principles, interaction patterns, and layout guidelines for a tool that's intuitive in the field and powerful at the desk.

Information Architecture

◎ Scout

Today's conditions, light score, camera settings. The default home view. Glanceable, real-time.

◐ Map

Interactive sun/moon/weather map with time scrubber. Mapbox GL with custom dark basemap.

⚡ Opportunities

7-day forecast scanner. Cards sorted by score. Push notification triggers. Pro feature.

◈ Planner

Shot planning timeline. Pick location + date, get minute-by-minute light windows.

◆ Spots

Community locations with EXIF data. Browse, save, contribute. Discovery engine.

Key UX Patterns

Progressive Disclosure
Surface the Light Score and character tag at the top level. Tap to expand the full condition breakdown (cloud layers, humidity, sun angle, etc).
Glanceable in Field
Critical info (score, settings, time) visible without scrolling. High contrast, large type for data values. Works with sunglasses on.
Time Scrubber Pattern
Horizontal scrubber on map and planner views. Dragging updates sun position, shadow casting, light score, and settings in real-time.
Opportunity Cards
Card-based layout for 7-day opportunities. Color-coded edge by type. Score badge top-right. Swipe to dismiss or save.
Pro Gating
Free users see blurred previews of pro content with a frosted glass overlay and gentle upgrade prompt. Never block, always tease.
Dark Basemap
Custom Mapbox style: desaturated terrain, subtle contours, minimal labels. Golden hour warm tint option. Never bright white map.

Accessibility Requirements

✓ WCAG 2.1 AA minimum
All text meets 4.5:1 contrast on dark surfaces. Large text (18px+) meets 3:1.
✓ Touch targets ≥ 44px
All interactive elements meet minimum touch target for outdoor/gloved use.
✓ Reduce motion support
Respect prefers-reduced-motion. Disable animations, map transitions become instant.
✓ Screen reader support
All data visualizations have aria-label with text equivalent.
✓ High contrast mode
Optional toggle increases border and text contrast.
✓ Color-blind safe
Never rely on color alone. All conditions have icon + label. Scores use shape + number.
