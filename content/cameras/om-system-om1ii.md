# OM System OM-1 Mark II

## Strengths

- 50fps with full AF tracking and a 120fps pro-capture mode captures moments impossible to time manually
- 2x crop factor doubles effective focal length, turning a 150mm lens into a 300mm equivalent for wildlife
- 8.5-stop IBIS is among the best available, enabling extraordinary handheld slow-shutter effects
- Compact, weather-sealed body with IP53 rating handles rain, dust, and freezing conditions
- Computational photography features including live ND, focus stacking, and high-res shot expand creative options

## Settings Notes

- **Best quality ISO range:** ISO 200-800 — clean output with good detail for the sensor size
- **Acceptable for print:** ISO 200-3200 — noise manageable for standard print sizes
- **Acceptable for web:** ISO 200-6400 — usable with noise reduction for screen output
