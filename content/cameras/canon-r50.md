# Canon R50

## Strengths

- Extremely lightweight and compact body is ideal for beginners and casual travel photography
- Affordable entry point into Canon's RF mirrorless ecosystem with upgrade path to full-frame
- 15fps electronic shutter provides respectable burst speed for the price class
- Dual Pixel CMOS AF II with subject detection delivers autofocus well beyond its price tier
- Intuitive interface with guided modes helps new photographers learn exposure and composition

## Settings Notes

- **Best quality ISO range:** ISO 100-400 — clean files with good detail
- **Acceptable for print:** ISO 100-1600 — reasonable quality for standard print sizes
- **Acceptable for web:** ISO 100-3200 — noise is manageable for screen-sized output
