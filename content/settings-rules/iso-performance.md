# ISO Performance by Sensor Size

Sensor size is the primary determinant of high-ISO noise performance. Larger sensors have larger photosites (pixels) that gather more light per pixel, producing cleaner images at higher ISO values. This guide provides practical ISO ranges for different sensor sizes at three quality tiers, helping you make informed exposure decisions in the field.

## Usable ISO Ranges by Sensor Size

### Full Frame (36 x 24mm)

| Quality Level | ISO Range | Notes |
|--------------|-----------|-------|
| Excellent (print/commercial) | 100-1600 | Virtually noise-free, full detail |
| Good (web/social, large prints with NR) | 1600-6400 | Mild luminance noise, easily cleaned in post |
| Usable (with noise reduction) | 6400-12800 | Visible noise, moderate detail loss after NR |
| Emergency only | 12800-51200 | Significant noise and color degradation |

### APS-C (23.5 x 15.6mm)

| Quality Level | ISO Range | Notes |
|--------------|-----------|-------|
| Excellent | 100-800 | Clean, detailed, no noise reduction needed |
| Good | 800-3200 | Light noise, handles NR well |
| Usable | 3200-6400 | Noticeable noise, plan for aggressive NR |
| Emergency only | 6400-25600 | Heavy noise, color shifts, detail loss |

### Micro Four Thirds (17.3 x 13mm)

| Quality Level | ISO Range | Notes |
|--------------|-----------|-------|
| Excellent | 100-400 | Clean images, no noise concerns |
| Good | 400-1600 | Mild noise, good after NR |
| Usable | 1600-3200 | Moderate noise, detail softening with NR |
| Emergency only | 3200-12800 | Heavy noise, significant quality loss |

### 1-inch Sensor (13.2 x 8.8mm)

| Quality Level | ISO Range | Notes |
|--------------|-----------|-------|
| Excellent | 100-200 | Clean at base settings |
| Good | 200-800 | Light noise appearing |
| Usable | 800-1600 | Noticeable noise, NR mandatory |
| Emergency only | 1600-6400 | Heavy noise, fine detail lost |

### Smartphone (typical 1/1.7" to 1/2.55")

| Quality Level | ISO Range | Notes |
|--------------|-----------|-------|
| Excellent | 50-100 | Clean in bright light |
| Good | 100-400 | Computational NR kicks in |
| Usable | 400-800 | Heavy processing artifacts |
| Emergency only | 800-3200 | Aggressive NR, painterly look |

## Approximate Sensor Size Advantage

Each step up in sensor size gains roughly 1-1.5 stops of ISO performance. This means:

| Comparison | Advantage |
|-----------|-----------|
| Full frame vs APS-C | ~1 stop (FF ISO 3200 ≈ APS-C ISO 1600 quality) |
| Full frame vs Micro Four Thirds | ~2 stops (FF ISO 6400 ≈ MFT ISO 1600 quality) |
| Full frame vs 1-inch | ~3 stops (FF ISO 6400 ≈ 1-inch ISO 800 quality) |

## Practical Guidelines

- **Landscape on tripod**: Always use base ISO (100). There is no reason to increase ISO when time is not a constraint.
- **Handheld general**: Stay at or below "Good" tier for your sensor size. Use wider aperture or slower shutter before raising ISO.
- **Action/sports**: Accept "Usable" tier ISO if needed to maintain fast shutter speed. A noisy sharp image is always better than a clean blurry one.
- **Astrophotography**: Push into "Usable" or even "Emergency" tier and stack multiple exposures to reduce noise statistically.

## Pixel Count vs ISO Performance

Higher megapixel sensors with the same physical size have smaller photosites and slightly worse per-pixel noise. However, when downsampled to the same output size, high-resolution sensors often match or exceed lower-resolution sensors because the additional pixels provide more data for noise averaging. In practice, the difference is minimal within the same sensor generation.
